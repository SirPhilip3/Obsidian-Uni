closures -> come funziona meccanismo di accesso gerarchico di accesso allo scope delle funzioni 

scope dipende dal momento in cui la funzione viene definita 

```js
let a = 1;
let b = 2;

let inc = function(v){
    let result = v;
    result = result + 1;
    return result;
}
```
