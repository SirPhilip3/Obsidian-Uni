+ Migliore performance e costo rispetto al [[Ring Network]] visto che abbiamo un *link* dedicato ad ogni *link* ( tra server e host ) ( dobbiamo avere solo $n-1$ *link* fisici )

**Cons** : 
+ Se fallisce il centro tutto il network muore
