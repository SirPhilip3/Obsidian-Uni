Un *NFA* è una quintupla $(Q\, \sum,\ \delta,\ q_0,\ F)$ dove : 
+ $Q$ è un insieme finito di *stati*
+ $\sum$ è un *alfabeto* finito
+ $q_0$ è lo stato *iniziale* $\in Q$
+ $F \subseteq Q$ stati *accettanti*
