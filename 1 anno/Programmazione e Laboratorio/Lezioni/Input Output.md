Includendo la libreria iostream

```c++
#include <iostream>

std::cin>> //input
std::cout<< //output
```

Oppure

```c++
#include <iostream>
using [[namespace]] std;

cin>> //input
cout<< //output
```


## Input

```c++
int a;
float b;
std::cin>>a;
std::cin>>b;
``` 

## Output
```c++
int a=3;
float c=324,324
cout<<a;
cout<<c;
```
